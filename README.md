# ai-postgresql

# AI PostgreSQL DBA

## Project Structure

Create a project folder.

```bash
mkdir ai_postgresql_dba
cd ai_postgresql_dba
```

Folder structure:

```
ai_postgresql_dba
│
├── ai_postgres_monitor.py
├── config.py
├── requirements.txt
└── reports/
```

Create reports directory:

```bash
mkdir reports
```
